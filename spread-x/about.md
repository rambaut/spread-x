# About SPREAD-X

SPREAD-X is a GIS-style application for mapping spatial phylogenetic trees
onto geographic maps. It uses D3 map projections to render layered geographic
data including base maps, GeoJSON overlays, point data, and phylogenetic trees.

## Features

- **Layer-based architecture** — Stack and reorder geographic layers
- **D3 map projections** — Natural Earth, Mercator, Orthographic, and more
- **Multiple data formats** — GeoJSON, TopoJSON, CSV points, phylogenetic trees
- **Interactive map** — Pan and zoom with mouse/trackpad
- **Visual settings** — Per-layer styling with live preview
- **Export** — SVG and PNG graphic export

## Framework

Built with [pearcore](https://github.com/rambaut/pearcore-lib) — a reusable
application framework extracted from [PearTree](https://peartree.live).
